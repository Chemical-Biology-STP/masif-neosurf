# Scripts Directory

This directory contains utility scripts for development, testing, and service management.

## Service Management

- `start_email_service.sh` - Start the email notification service
- `stop_email_service.sh` - Stop the email notification service
- `check_email_service.sh` - Check email service status

## Testing Scripts

- `test_ssh_connection.py` - Test SSH connection to HPC
- `test_email_notification.sh` - Test email functionality
- `test_module_load.py` - Test module loading on HPC
- `test_slurm_script.py` - Test SLURM script generation
- `test_final_module.py` - Test final module configuration

## Debug/Discovery Scripts

- `find_*.py` - Scripts to locate various system components
- `check_*.py` - Scripts to verify configurations
- `debug_*.py` - Debugging utilities
- `verify_module.py` - Module verification

These scripts were used during development and deployment setup.
